# Usage

## Create repository

1. Navigate to the repos page

<img src="repos.png" alt="repos"/>

2. Click on "Create Repository" to open the create dialog. You can select the type of the repository you want to create 
    And if the repository will be public available without authentication.

<img src="create-repo.png" alt="create-repo"/>

3. The repository is now available

<img src="repo-created.png" alt="repo-created"/>

## Create service account

A service account is used to authenticate at a repository

1. Navigate to the service account page

<img src="service-account.png" alt="service-account"/>

2. Click on "Create Service Account" to open the create dialog.
    You need to specify a name, expiration date and you can select the permissions for each repository.

<img src="create-service-account.png" alt="create-service-account"/>

3. The service account is now available. The secret will be shown once.
